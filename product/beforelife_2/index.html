예전의 난 개발기 - 2

Oct 11 2020

환경 구축#

사용중인 AWS EC2(t3.medium)에 docker를 이용해 mysql을 설치했다.

  • docker 설치 및 확인
$ curl -fsSL https://get.docker.com/ | sudo sh
$ docker --version
  • docker mysql 설치
$ docker pull mysql:[version]
$ docker images
$ docker run -d -p 3306:3306 -e MYSQL_ROOT_PASSWORD=[password] --name [container-name] -v [path-to-data]:/var/lib/mysql mysql:[version]
  • database 생성
$ docker exec -it [container-name] bash
# mysql -uroot -p
> create database [database-name];
> create user 'username'@'%' identified by 'password';
> grant all privileges on [database-name].* to 'username'@'%';
> flush privileges;

EC2 보안 그룹 인바운드 규칙에 3306(mysql), 8000(django) 정도를 추가한다.
