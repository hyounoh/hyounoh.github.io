예전의 난 개발기 - 1

Oct 9 2020

시작#

가끔 짐 정리를 하다 예전에 썼던 일기를 발견할 때가 있다.
그 내용은 매번 새롭게 느껴지지만, 일기를 읽고 난 후의 행동은 항상 같다.
다시 일기 쓰기.

일기를 다시 쓰다 보면 예전의 난 어떤 생각을 하고 살았는지 궁금해진다.
같은 시간 어제의 난 어떤 생각을 했는지, 작년의 10월 9일엔 어땠는지.

IA#

  • 일기 쓰기
  • 일기 보기
    • 오늘과 관련된 일기 목록 (요일, 일, 월)
  • 소셜 로그인 (Kakao, Google)

Stack#

  • Client
    • Flutter
  • Server
    • Django Rest Framework
  • Database
    • MySQL

Camera, Bluetooth 등의 Device 기능이 필요하지 않고, GL 등의 고성능 처리가 필요하지 않기 때문에 Flutter를 이용한 cross-platform 개발을 한다.

서비스를 구성하는 객체들의 명료한 modeling이 가능하면 Django로 개발한다.

형태소 분석, aggregation 등의 기능이 필요하지 않기 때문에 MySQL을 사용한다. Session 처리는 Django를 이용한다.
